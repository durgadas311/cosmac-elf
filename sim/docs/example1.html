COS/MAC ELF Example 1

This is the first example in the August 1976 Popular Electronics article. It monitors the IN button and makes the Q LED follow the current state of that switch.

The program logic is: 

    START: REQ
LOOP:  BN4 START
       SEQ
       BR  LOOP

To run this, create a config file (plain text) that contains this line: 

    prom = example1.bin

Where "example1.bin" is the file in this repository under "progs/example1.bin".

Then invoke the simulator with the command: 

    java -jar VirtualCOSMAC_ELF.jar <config-file>

Once the simulator starts, click the PROM jumper to set it "on", then switch off LOAD and switch on RUN. At this point, depressing the IN button should cause the Q LED to light, and releasing it should cause the LED to go dark.
