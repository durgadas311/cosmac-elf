IDIOT/4 Monitor Demo

IDIOT/4 is a CDP1802 Monitor program that works on many ELF-style computers. IDIOT/4 is discussed here. The versions in progs/idiot4/ are identical to the latest from those located here, except that they are compiled to run at location 0000. The demo uses progs/idiot4/idiotX0.bin, which matches the default configuration of the simulator serial port.

The following configuration may be used to run it: 

ram = 64k
quart
quart_att = "TelnetSerial 192.168.0.0 12345"
preload = idiotX0.bin
autorun

Replace "192.168.0.0" with the hostname or IP address of your system, and replace "12345" with your favorite port number. Once the simulator has been started, you may connect to the IP and port using 'telnet' or your favorite program that supports telnet or TCP/IP socket connections.

Whatever program you use to connect needs to run in a "raw" or "character" mode, to avoid lines getting buffered and CR being translated. For telnet, the command "mode character" will suffice. This command may also be put in a ~/.telnetrc file to avoid having to type it each time telnet is started.

You will need to press CR (carriage return - i.e. ENTER on modern keyboards) in order to let IDIOT/4 calculate the serial baud rate and start.
