IDIOT/4 Monitor Demo

IDIOT/4 is a CDP1802 Monitor program that works on many ELF-style computers. IDIOT/4 is discussed here. The versions in progs/idiot4/ are identical to the latest from those located here, except that they are compiled to run at location 0000. The demo uses progs/idiot4/idiotX0.bin, which matches the default configuration of the simulator serial port.

The following configuration may be used to run it: 

ram = 64k
quart
quart_att = TeletypeSerial
preload = idiotX0.bin
autorun

You will need to press CR (carriage return - i.e. ENTER on modern keyboards) in order to let IDIOT/4 calculate the serial baud rate and start.
