Simulator Operation

The simulator features are operated from the menu bar. The following menu items are available:

Debug

Trace (custom)
Pops up a dialog for tracing the CPU execution of instructions. Trace output is sent to stderr or the "log" file. Tracing may be canceled with the "Trace OFF" menu item. Tracing is also canceled by changing the tracing parameters.
Trace ON
Turns tracing on immediately, flooding the log/stderr with trace lines. Tracing may be canceled with the "Trace OFF" menu item.
Trace OFF
Cancels any tracing currently enabled.
Dump To Log/GUI
Alternates between dumping data to log file (stderr) or pop-up windows. This does not affect tracing output.
Dump CPU
Dumps information about the CPU state.
Dump Machine
Dumps information about the platform.
Dump Core
Dumps the contents of RAM to a file.
Load Prog
Fills RAM with the contents of a file.
Dump Page
Dumps a 256-byte page of the CPU address space (ROM or RAM as appropriate).
Disassemble
Dumps disassembled instructions from a range page of the CPU address space.
Copy ROM to RAM
Copys the contents of the PROM into RAM. Does not require the PROM to be enabled (via jumper plug). This is essentially the same as "Load Prog" where the selected file is the one specified by the "prom" property.
Dump ELF-FP
Dumps information about the standard ELF front panel hardware.
Dump HEXKEYPAD
Dumps information about the hexadecimal keypad. Menu item does not exist if keypad is not installed.
Dump PIXIE
Dumps information about the PIXIE graphic display hardware. Menu item does not exist if pixie is not installed.
Reset
Performs a hardware reset. This should be equivalent to setting the RUN switch off.
Quit
Ends the simulation. Clicking the "close" button on any of the windows also ends the simulation.
