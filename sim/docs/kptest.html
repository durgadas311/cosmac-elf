COS/MAC ELF Keypad test

This program confirms proper operation of the hex keypad (as described in Popular Electronics), Using a CD4515 connected to N2. See "progs/kptest.lst" for logic.

To run this, create a config file (plain text) that contains this line: 

    hexkeypad
prom = kptest.bin

Where "kptest.bin" is the file in this repository under "progs/kptest.bin".

Then invoke the simulator with the command: 

    java -jar VirtualCOSMAC_ELF.jar <config-file>

Once the simulator starts, click the PROM jumper to set it "on", then switch off LOAD and switch on RUN. At this point, depressing a button on the keypad should result in that hex value being displayed.
