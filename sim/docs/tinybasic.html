TMSI Tiny BASIC Demo

A copy of TMSI Tiny BASIC is in progs/tiny_basic/tb0_test.bin. TMSI Tiny BASIC is discussed here, and can be obtained here.

The following configuration may be used to run it: 

ram = 64k
quart
quart_att = TeletypeSerial
quart_qmark=0
quart_efmark=0
preload = tb0_test.bin
autorun

You will need to press CR (carriage return - i.e. ENTER on modern keyboards) in order to let Tiny BASIC calculate the serial baud rate and start.

See the documentation at these URLs for information on how to run Tiny BASIC.
