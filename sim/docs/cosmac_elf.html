COS/MAC ELF Help

Basic operation of simulator features is described here.

The simulator is customized using properties that are specified by "property=value" expressions. These assignments may be in a plain-text file whose filename is given on the command line (or assumed), and/or by assignments as command line arguments. Details of properties are here.

Without a configuration file or properties, an ELF with 256 bytes of RAM is created, with no optional hardware.
